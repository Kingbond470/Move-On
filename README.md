#### This app is a Packer and Movers used to Deliver the Baggage during a Move on to a New location.

## Technologies
* Recycler View 
* Firebase Auth 
* Lottie Animation
* Google Map Api 
* Kotlin
* Paymet GateWay
* Room DataBase
* Material Components

## Features
* Users can Select location on the map to Deliver the Items to a location.
